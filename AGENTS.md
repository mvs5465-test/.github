# AGENTS.md

Instructions for human + AI contributors in this repository.

## Product

- `dot-github` is the shared org-level standards repo for GitHub conventions and repository setup guidance.
- Right now it is documentation-first rather than workflow-heavy.

## Structure

- `README.md` is the short repo entrypoint.
- `REPO_SETTINGS_CHECKLIST.md` is the main practical reference.

## Working Rules

- Keep this repo focused on org-wide standards, not project-specific behavior.
- Prefer concise, actionable guidance over long policy prose.
- If workflow templates or reusable docs are added later, keep the repo scoped to shared GitHub operations.

## Verification

- For docs changes, verify links and filenames remain accurate.
